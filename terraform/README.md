## terraform

AWS リソースを Terraform で管理します。

| ファイル      | 説明                   |
| ------------- | ---------------------- |
| provider.tf   | AWS provider 設定      |
| variables.tf  | Terraform 変数         |
| iam.tf        | Lambda 実行用 IAM Role |
| lambda.tf     | Lambda Function 定義   |
| apigateway.tf | API Gateway 定義       |
| output.tf     | API URL 出力           |
